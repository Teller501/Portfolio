Projects

Superhero project
Language: Java
The project allows the user to create, edit, search and delete superheroes in a database based on the CRUD principle (Create, Read, Update, Delete)

We started this project at the beginning of the semester to learn how to use ArrayList in Java. Later we had to develop it to be able to read from and save to a CSV file.
Adventure game
Language: Java
The project is a text-based adventure game, inspired by 1977's Colossal Cave Adventure by William Crowther. The player is able to navigate between different rooms, pick up items, attack enemies, etc.

Our learning objectives for the project were Object-Oriented Programming, including object references, inheritance and polymorphism.
Project 'Delfinen'
Language: Java
'Delfinen' is a swimming club for which we had to create an administrative system to manage membership information, quotas, and swimmer results. It also follows the CRUD principle as the Superheroes project.

We simulated that we had a real customer, who we had to meet with weekly to keep them updated, and possibly adjust some requirements for the program.
